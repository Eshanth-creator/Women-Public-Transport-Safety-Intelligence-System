# Women Public Transport Safety Intelligence System

## Project Overview
The Women Public Transport Safety Intelligence System aims to enhance the safety of women using public transportation. It integrates various technologies to provide real-time information and support to ensure safer travel experiences.

## Features
- **Real-time tracking**: Monitor buses and other public transport vehicles.
- **Emergency SOS**: Quick access to emergency contacts and services.
- **User feedback system**: Allow users to report safety issues or incidents.
- **Data analytics**: Analyze travel patterns and incidents for improvements.

## Technical Stack
- **Frontend**: React.js for building the user interface.
- **Backend**: Node.js with Express for server-side logic.
- **Database**: MongoDB for data storage.
- **APIs**: Integration with third-party APIs for real-time transport data.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Eshanth-creator/Women-Public-Transport-Safety-Intelligence-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Women-Public-Transport-Safety-Intelligence-System
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
